Магистральный последовательный интерфейс

Магистральный последовательный интерфейс
Это совокупность технических средств и правил, которые обеспечивают обмен информацией между абонентами интерфейса последовательным кодом по общей информационной магистрали. Требования к реализации интерфейса описаны в ГОСТ Р 52070.
ГОСТ Р 52070-2003
Государственный стандарт Российской Федерации, который называется «Интерфейс магистральный последовательный. Системы электронных модулей. Общие требования». Документ распространяется на магистральный последовательный интерфейс с централизованным управлением. Область применения стандарта — системы электронных модулей, где интерфейс используется для обмена информацией между различными блоками. Аналогом является стандарт MIL-STD-1553B.

Использование программы

Данная программа позволяет упростить работу с командными словами и словами данных, предоставляя инструменты для написания масок битов переменных для их последующей распарски или упаковки, а так же для создания командных слов и их расшифровки. Программа не поддерживает команды управления и предполагает, что ваш протокол используется для приемо-передачи данных.
