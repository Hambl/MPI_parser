Пакет данных

Эта программа позволит работать сразу со всем пакетом данных в зависимости от командного слова.

  1. Укажите настройки, введите командное слово и нажмите на кнопку "Создать таблицу".
  2. Объяедините ячейки таблицы под ваши переменные и нажмите на кнопку "Заполнить переменные".
  3. Введите названия переменных в необходимые места. Если в пакете присутсвуют пустые биты, оставтье их пустыми.
  4. Далее нажмите на кнопку "Рассчитать маски"

Для более подробного руководства обратитесь к справке .

Примечание:

Нумерация слов данных в таблице соответвует индексу массива (т.е. начинается с нуля).

Настройка

Формат маски переменной DW[dw]_[var]_Msk [msk]
Формат маски сдвига битов DW[dw]_[var]_Pos [msk]

Изменение масок

Введите формат масок используя любые символы по вашему усмотрению. Используейте следующие обозначения:
[dw] - для указания номера слова данных,
[var] - для вставки имени переменной,
[msk] - для вставки рассчитанной маски битов.
