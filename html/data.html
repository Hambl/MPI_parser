Слово данных

Эта программа позволяет автоматически рассчитать маски переменных и их позиций в слове данных по битам для последующего использовния в распарске или упаковке данных.

  1. Укажите какие маски вы хотите рассчитать и измените формат генерируемых масок по своему усмотрению.
  2. Укажите количество переменных в слове данных.
  3. Заполните поля переменных и битов.
  4. Нажмите на кнопку "Рассчитать", чтобы сгенерировать маски.

Для более подробной информации обратитесь к справке .

Примечание:

Для указания пустых битов используйте имя переменной Empty . Маски для пустых переменных генерироваться не будут.

Формат маски

Формат маски переменной [var]_Msk [msk]
Формат маски сдвига битов [var]_Pos [msk]

Изменение масок

Введите формат масок используя любые символы по вашему усмотрению. Используйте обозначения [var] для вставки имени переменной и [msk] для вставки рассчитанной маски битов.
