Командное слово

Эта программа позволит разобрать командное слово на составляющие, или наоборот собрать его из адреса, направления передачи, подадреса и количества слов данных.

  1. Выберите систему счисления с которой вам удобно работать.
  2. Корректно заполните поля данных. Для сборки командного слова вы можете использовать кнопки для быстрого вода некоторых значений.
  3. Нажмите на кнопку "Разобрать" (в случае разбора командного слова) или "Собрать" (в случае сборки командного слова из составляющих).

Для более подробной информации обратитесь к справке.

Примечание:

В соответсвии с ГОСТ Р 52070 значение числа СД 00000 соответсвует маскимальному количеству слов данных - 32.
